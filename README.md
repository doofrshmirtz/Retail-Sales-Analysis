Retail Sales Analysis (Tonin Kaipada Thomas, M.Sc Data Science)

This project analyzes retail sales data across multiple stores and products using Python.
1. Tools Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

2. Dataset:
The dataset contains-
- Item ID : Product Indetifier
- Store ID : Store Identifier
- Monthly sales : Time period of sales
- Units sold

Two datasets are included-
- Raw sales data
- Processed long-format dataset

3.Analysis Performed:
1. Loading and inspecting the raw dataset
2. Reshaping the dataset from wide format to long format
3. Handling missing values
4. Converting date columns to datetime format
5. Aggregating sales data by time, store, and product
6. Identifying top-performing stores
7. Identifying top-selling products
8. Visualizing trends and performance using charts
